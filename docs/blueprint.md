# **App Name**: MyRepublic Malang Leads

## Core Features:

- Lead Capture Form: A simple form to capture potential customer information (name, contact details, address).
- Firestore Integration: Store lead data securely in Firestore for follow-up.
- Exclusive Offer Display: Highlight special MyRepublic offers available only in Malang.
- Area Availability Check: Allow users to check MyRepublic service availability at their specific address in Malang.
- FAQ Section: Address common questions about MyRepublic services in Malang.
- AI-Powered Offer Customization Tool: Dynamically tailors MyRepublic plan recommendations to individual users, based on provided input such as address or internet use behavior, to optimize appeal.
- Coverage Area Integration from KMZ (GeoJSON): Enables a realistic 'Check Coverage' feature on /register and /coverage-check routes, using geographic data to determine service availability.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to reflect MyRepublic's branding and evoke trust and reliability.
- Background color: Light gray (#F0F0F0), providing a clean and modern look.
- Accent color: Teal (#008080) for buttons and highlights, complementing the primary blue and adding a touch of sophistication.
- Body and headline font: 'PT Sans', a humanist sans-serif, for readability and a modern feel.
- Use clean, line-style icons to represent different MyRepublic services and features.
- Clean and responsive layout optimized for lead capture on all devices.
- Subtle animations on form elements to guide user interaction and enhance engagement.
